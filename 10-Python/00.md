# Python Environment Setup

This guide covers how to set up a Python virtual environment and install dependencies.

Reference: [The Python Code Example Handbook](https://www.freecodecamp.org/news/the-python-code-example-handbook/)

## Create an Environment

Create a project folder and a `.venv` folder within it:

```bash
# macOS/Linux
mkdir myproject
cd myproject
python3 -m venv .venv
```

```bash
# Windows
mkdir myproject
cd myproject
python -m venv .venv
```

## Activate the Environment

Before you work on your project, activate the corresponding environment:

```bash
# macOS/Linux
. .venv/bin/activate
```

```bash
# Windows
.venv\Scripts\activate
```

Your shell prompt will change to show the name of the activated environment.

## Check if You Are Running in a Virtual Environment

```bash
pip -V
```

Reload your shell profile if needed:

```bash
source ~/.bash_profile
```

## Install Dependencies from a Requirements File

```bash
pip3 install -r requirements.txt
```
