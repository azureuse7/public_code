# Absolute and Relative Paths
> Understanding how file paths work is fundamental to navigating Linux, writing scripts, and configuring tools. An **absolute path** always starts from the root `/`, while a **relative path** starts from the current working directory.

The top directory is the root directory and is represented as `/`

**absolute** **path**: /sam/tom/jerry

**Relative** **path**: Jerry (notice no /)
